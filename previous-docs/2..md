# 2. 뉴비 스킬 플랜

## 뉴비 여러분들을 위한 스킬플랜 안내

1SOBR에 들어온 너의 첫번째 목표는 플릿이 어떤식으로 굴러가는지 파악하는것임. 스킬이 낮은 사람도 충분히 제 가치를 할 수 있는게 이브의 플릿전임.

우선적으로 너가 타게 될 배는 민마타 프리깃인 Slaser이고 정찰용 함선임. 그후 타게될배는 너가 타고 돈을 벌어야할 Vexor Navy issue.

아래 스킬중 기본스킬이 아니어서 돈을 주고 배워야한다면 디렉터에게 문의해서 구입할것!

스킬 플랜대로 스킬을 치면서 탈수있는 배를 하나하나 늘려나가는게 너의 목표.

아래는 임플란트라고 캡슐에 박으면 스킬치는 속도 빨라지는것들임. 캡슐터지면 같이 날라가니 조심

```text
Ocular Filter - Basic
Memory Augmentation - Basic
Neural Boost - Basic
Cybernetic Subprocessor - Basic
```
